
### Vue 2 CRUD Application - Laravel


> This project contains

1. Create contacts
2. Delete contacts
3. Update contact
4. view all the contacts
5. Saved data into database

>Uses Guideline

> clone the repo,

> move to vuecrud directory

> composer install or update

> npm install

> php artisan key:generate

> setup .env file

``` setup database ```

> php artisan migrate

> php artisan serve
